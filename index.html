O HTML (Hypertext markup language) é uma linguagem de marcação fundamental para a construção de sites, criando o conteúdo e a estrutura principal dos sites

O CSS (Cascading Style Sheet) é uma linguagem de estilização, usada em conjunto com o HTML para modificar diversos aspectos visuais dos elementos

O Java Script é uma linguagem de programação utilizada principalmente para a criar interatividade em páginas da web, como botões, por exemplo
