O HTML é uma linguagem de marcação, o que significa que ela marca elementos para receberem certas propriedades através do CSS

Tags de Texto

Headings

As headings são tags representadas por <hx>, onde x é um número, e são usadas principalmente para fazer textos de títulos e subtítulos, elas vão do h1 ao h6 e, quanto maior o número, menor o tamanho da fonte

Paragraph

A tag paragraph é representada por <p> e é usada para escrever parágrafos comuns em textos

Break

A tag break é representada por <br> e faz o papel da tecla "enter" do teclado, ou seja, quebra uma linha, fazendo com que o texto continue na linha abaixo

Ordered e Unordered Lists

Representadas respectivamente por <ol> e <ul>, essas tags são usadas para construir listas, sendo a <ol> usada para listas onde a ordem importa e a <ul> para listas onde a ordem não importa. Cada elemento dessas listas deve estar dentro de uma tag <li>

note que, em ol's, os elementos serão automaticamente enumerados, enquanto em ul's eles terão bullet points(•) antes de cada elemento

Outras Tags

Script

A tag script é usada para conectar um arquivo JavaScript à um arquivo HTML

Body

A tag body é a tag que define o corpo de um projeto, ou seja, tudo que será exibido para o usuário

Hypertext Reference

Hypertext reference ou referência de hipertexto, nada mais é do que um nome diferente para link, que é exatamente a função desta tag. Representada por <href>, essa tag coloca um link em um fragmento de texto, podendo levar à um arquivo local ou à um link externo, essa tag deve ser usada dentro de uma tag de texto, como <p>, por exemplo

Div

A tag div é utilizada para criar seções ou divisões na página, com a ajuda do css, ela também pode se tornar um contâiner, com cores, formato e etc.

Button

A tag button é usada para criar botões que podem ser configurados usando o JavaScript

Class

A tag class serve para atribuir uma classe à outra tag, ou seja, um nome que diferencia uma tag de outras do mesmo tipo. Assim, o css também diferenciará essas tags e poderá atribuir formatações diferentes à duas tags semelhantes
